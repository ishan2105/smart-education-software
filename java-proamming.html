Elaborated Notes

1. Java Syntax and Structure

  • Class Structure: public class Main { public static void main(String[] args) { ... } }
  • Statements: End with a semicolon ;.
  • Case Sensitive: Java is case-sensitive.

2. Variables, Data Types, and Operators

  • Data Types: int, float, double, char, boolean, String.
  • Operators: Arithmetic (+, -, *, /), Relational (==, !=, <, >), Logical (&&, ||, !).

3. Control Flow

  • Conditional: if, else if, else, switch-case.
  • Loops: for, while, do-while.

4. Methods and Scope

  • Method Declaration: int add(int a, int b) { return a + b; }
  • Scope: Local and global variables.

5. Arrays and Strings

  • Arrays: int[] arr = new int[5];
  • Strings: String s = "Hello";

6. Object-Oriented Programming

  • Classes & Objects: class Car { ... }
  • Encapsulation: Using private/public members.
  • Inheritance: class Child extends Parent { ... }
  • Polymorphism: Method overloading and overriding.

7. Collections Framework

  • List, Set, Map: ArrayList, HashSet, HashMap.
  • Iterators: Used to traverse collections.

8. Exception Handling

  • Try-Catch: try { ... } catch(Exception e) { ... }
  • Finally: Code that always runs after try/catch.

9. File I/O

  • Reading/Writing: FileReader, FileWriter, Scanner, BufferedReader.
